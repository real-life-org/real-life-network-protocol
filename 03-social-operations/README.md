# Social Operations

Kernoperationen, durch die das Netzwerk wächst.

Startpunkt: Abschnitt 7 und 8 in [../real-life-network-protocol.md](../real-life-network-protocol.md).

Jede Operation soll mindestens beschreiben:

- Zweck
- Inputs
- Ablauf
- Outputs
- nächste Schritte
- App-Unterstützung
- Agentenunterstützung
- Metriken
- Risiken
