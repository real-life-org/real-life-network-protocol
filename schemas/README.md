# Schemas

Maschinenlesbare Schemata für soziale Operationen, RLS-Item-Views, Quests, Relationen und Rollout-Artefakte.

Die Schemata in diesem Repository sollen backend-agnostische RLNP-Views wie Quests, QuestRuns, Confirmations und Relationen beschreiben. Sie sollen keine alternativen WoT-Formate für signierte Verifikationen oder Attestations definieren. Dafür gilt die `wot-spec`: WoT Trust 001/002 als VC-JWS. Dieses Repository darf darauf verweisen und App-Projektionen beschreiben.
